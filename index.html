☁ Cloud Sync

Share one live copy of your books across all devices and your team.

First-time setup (one person, ~10 min):
1. Go to console.firebase.google.com and create a free project.
2. Build → Authentication → Get started → enable Email/Password.
3. Build → Firestore Database → Create database (Production mode).
4. In Firestore Rules, paste:
allow read, write: if request.auth != null; then Publish.
5. Project settings (⚙) → Your apps → Web (</>) → register → copy the firebaseConfig object and paste it below.
6. Pick one shared email + password for the team and Connect. Give the same config + login to teammates.

Tip: set up from the device that already has your real data — it seeds the cloud the first time.
